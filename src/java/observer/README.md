# 观察者模式（行为型模式）  

## 介绍

什么情况下使用？解决了什么问题？  

- 定义对象间的一种一对多的依赖关系，当一个对象的状态发生改变时，所有依赖它的对象都得到通知并且自动更新
- 一个对象改变给其他对象通知的问题，而且要考虑到易用和低耦合，保证高度的协作
